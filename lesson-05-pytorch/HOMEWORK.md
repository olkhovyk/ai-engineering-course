### 4. Розберись у коді

Код `nano_gpt.py` прокоментований українською і відповідає крокам у візуалізаторі `gpt_visualizer.html`. Відкрий візуалізатор у браузері — він покаже кожен крок графічно.

| Крок у візуалізаторі | Де в коді |
|---|---|
| 1. Токенізація | `chars`, `stoi`, `itos`, `encode`, `decode` |
| 2. Embeddings | `self.tok_emb + self.pos_emb` |
| 3. Self-Attention | `Q, K, V → scores → softmax → weighted sum V` |
| 4. Multi-Head | `view(B, T, N_HEAD, head_dim).transpose` |
| 5. Causal Mask | `register_buffer("mask", tril(...))` |
| 6. Transformer Block | `Block.forward` |
| 7. Генерація | `NanoGPT.generate()` |
